Ride Alarmer

One page sends GPS updates from the rider's phone. The other page listens, measures distance to your destination pin, and rings when the cab enters your radius.

Open tracking page Open monitoring page
  • No backend server
  • Firebase Realtime Database
  • Leaflet + OpenStreetMap

How it works

  1. The rider opens the tracking page and allows location access.
  2. The page writes coordinates to Firebase every few seconds.
  3. You set a destination pin on the monitor page.
  4. The alarm fires when the rider is within range.

Setup checklist

  1. Create a Firebase project and enable Realtime Database.
  2. Paste the Firebase config into firebase-config.js.
  3. Host these static files on GitHub Pages or Netlify.
  4. Share the tracking link with the rider.
